# Tributo Kentaro Miura!

Este é meu primeiro projeto partindo do absolutamente do 0.  Foi um exercício do curso B7Web realizado em setembro de 2023. 


# Conteúdo

O exercicio consistia em projetar uma página de tributo para homenagear algem que já faleceu. Ecolhi o autor de mangás "Kentaro Miura" criador de uma das obras que mais admiro, e que sem dúvida mudou o genero de Dark Fantasy para sempre.
